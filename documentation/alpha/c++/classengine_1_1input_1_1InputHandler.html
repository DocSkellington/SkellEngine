Detailed Description

Handles the input with the user such as window resized, key pressed, etc.

  • Input is used for player-generated inputs. That is, when the player presses a button, moves the mouse, etc. Typically, it's a sf::Event object.
  • Event is used for the events (engine::events::Event) sent by the input handler through the event handler when a received input matches a registered input.

Typically, the connections are loaded from a JSON file describing the input
