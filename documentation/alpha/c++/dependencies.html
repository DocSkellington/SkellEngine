Building

The library files are needed for these dependencies. Please, see the web sites to know how to compile or download these files for your operation system.

  • SFML 2.5 distributed under zlib/png license.

The following dependencies are given with the source code:

  • A modified version of tmx-lite distributed under zlib/png license.
  • TGUI distributed under zlib/png license.
  • Thor distributed under zlib/png license.
  • Sol3 distributed under MIT license.
    • Sol3 proposes a way to compile Lua. To ease Windows development, we use this to build the Lua library. Lua is distributed under MIT license. Lua version can be set with the CMake parameter SOL2_LUA_VERSION. Lua5.2+ is recommended (Lua5.1 has some performance issues and lacks some useful functions).
  • JSON for modern C++ distributed under MIT License.
  • Catch2 distributed under Boost Software License 1.0.
