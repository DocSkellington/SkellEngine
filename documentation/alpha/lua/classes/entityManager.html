Class entityManager

Entity Manager.

The entity manager handles the entity of the engine. One can add, remove and get entities. It can be accessed as game.entityManager

Methods

entityManager:addEntity (type) Creates a new entity.
entityManager:addEntity (type, components) Creates a new entity with the given components.
entityManager:getEntity (type) Gets an entity.
entityManager:getEntity (type, components) Gets an entity that has at least the given components.
entityManager:removeEntity (entity) Removes an entity from the game.


Methods

entityManager:addEntity (type)
Creates a new entity.

The new entity does not have any components and is not added to the systemManager. The user must manually do it (after the components have been defined) for the entity to be used.

Parameters:

  • type The type of the entity to construct and add

Returns:

    The new entity
entityManager:addEntity (type, components)
Creates a new entity with the given components.

The new entity is added to the systemManager. So, don't forget any components in the list.

Parameters:

  • type The type of the entity to construct and add
  • components The table describing the components and their fields

Returns:

    The new entity
entityManager:getEntity (type)
Gets an entity.

The first matching entity is returned

Parameters:

  • type The type of the entity to get

Returns:

    The first matching entity
entityManager:getEntity (type, components)
Gets an entity that has at least the given components.

The first matching entity is returned. The entity can have more components than the listed ones

Parameters:

  • type The type of the entity to get
  • components The list of the names of the comonents the entity must have

Returns:

    The first matching entity
entityManager:removeEntity (entity)
Removes an entity from the game.

It also removes the entity from the systems. Therefore, the entity is completely removed from the engine.

Parameters:

  • entity The entity to remove

Returns:

    Whether the entity could be removed or not
